# Glock Grotesk 2.0 Font Files

Download Glock Grotesk 2.0 from: https://www.tsankotype.com/shop-4

Place the following font files in this directory:

- `GlockGrotesk-Regular.woff2`
- `GlockGrotesk-Regular.woff`
- `GlockGrotesk-Medium.woff2`
- `GlockGrotesk-Medium.woff`
- `GlockGrotesk-Bold.woff2`
- `GlockGrotesk-Bold.woff`
- `GlockGrotesk-Black.woff2`
- `GlockGrotesk-Black.woff`

The app will fall back to system fonts if these files are not present.
